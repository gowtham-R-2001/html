Stock entry

Bill panel

  1. Overview
  2. Language : Java
  3. Tools : Notepad++
  4. Compiler : JVM JRE
  5. Concepts : GUI, SQLITE, JDBC, Arrays, regex etc.

Description
    This is a student database system for schools. It is based on the CRUD operations. Creation, Retrival, Updation, Deletion. It includes four section.
  1. Input panel
  2. Display panel
  3. Edit panel
  4. Delete panel

Input Panel
    In this panel, the roll number will be generated automatically starts from 101 or shall be modified. It asks you to enter the student name, five subjects marks of student. Then, the total marks will be calculated automatically. The submit button performs a action, stores the details of a student in the database. The revert button revrt the changes. This is a creation operation from the CRUD operations.
Display Panel
    In this panel, it asks you to enter the roll number of a particular student. The showgrade button performs a action, it displays the details, marks and also grade for that particular student. The data is retrieved from the database. This is a retrieval operation from the CRUD operations.
Edit panel
    It allows you to directly edit the details of a student except the roll number. After editing the data of the student, it will automatically gets updated in the database at the backend. This is a updation operation from the cRUD operations.
    It asks you to enter the roll number of a student that you wish to delete. After entering the roll number it displays the data of that student. For confirmation, it asks you to press the press the 'delete' button. This is a deletion operation fom the CRUD operations. Therefore, from these four panels it covers the all the 4 CRUD operations.
